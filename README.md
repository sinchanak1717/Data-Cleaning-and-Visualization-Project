# Data Cleaning and Visualization Project

## Description
This project demonstrates data cleaning and visualization using the Titanic dataset. It includes handling missing values, removing duplicate records, and creating visualizations such as a histogram, box plot, and correlation heatmap to better understand the data.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
- Titanic Dataset (titanic.csv)

## Output
- Cleaned dataset (`cleaned_titanic.csv`)
- Histogram
- Box Plot
- Correlation Heatmap
